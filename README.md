# SPACE-JUMP

Three.js で制作した横スクロール型ジャンプアクションゲームです。
スパイクを飛び越えながら距離を伸ばすことを目指します。

## 操作

| 操作 | 動作 |
|------|------|
| `Space` / `↑` | ゲーム開始 / ジャンプ |
| クリック / タップ | 同上 |

## 技術スタック

- [Three.js r128](https://threejs.org/) — 3D レンダリング
- Canvas 2D API — トランジション描画
- Google Fonts (Orbitron) — HUD フォント

## カスタマイズ

コード上部の定数で挙動を調整できます。

| 定数 | デフォルト | 説明 |
|------|-----------|------|
| `GRAVITY` | `0.000075` | 重力 |
| `JUMP_SPEED` | `0.02` | ジャンプ初速 |
| `SPIKE_SPEED` | `0.015` | スパイクの速度 |
| `SPIKE_NUM` | `4` | 同時出現数 |
| `SPIKE_MIN_GAP` / `SPIKE_MAX_GAP` | `5` / `20` | スパイク間隔の範囲 |
